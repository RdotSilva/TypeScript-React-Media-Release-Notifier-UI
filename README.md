# TypeScript React Media Release Notifier UI

This is the UI for the Media Release Notifier. It will require the backend server to be running (still a WIP)

## Installation

1. Install dependencies in main project folder.

```
npm install
```

Run the client

```
npm start
```

## Built With

- TypeScript
- React
- Chakra UI
